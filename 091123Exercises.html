Following are javascript intermediate concepts:

This: The "this" keyword in JS is notoroious for being difficult to explain. It (this) refers to diffferent things depending on its execution context.

Object oriented Programming: (oop); A programming paradigm that uses objects and classes for organizing code. It is structured around four main principles: encapsulation, abstraction, inheretance, and polymorphism.

Classes: A blueprint for creating objects with shared properties and methods. Classes provide a more structured and objecct-oriented -like approach to designing code.

Constructor Method: A special method in a class that gets called automatically when an instance of the class is created. It usually initalize object properties.
